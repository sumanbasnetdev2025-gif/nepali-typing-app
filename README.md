# नेपाली टाइपिङ (Nepali Typing App)

A professional Nepali Unicode typing web app built with Next.js 14 + Tailwind CSS.

## Features
- Romanized → Devanagari transliteration
- 33+ Nepali Unicode fonts
- Bold / Italic rich text editor
- Portrait & Landscape document layout
- PDF export with 12 texture backgrounds (Lokta paper, Parchment, etc.)
- Word suggestions with dictionary ranking
- Preeti → Unicode converter

## Getting Started
\`\`\`bash
npm install
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS v4
- TypeScript
- Custom transliteration engine
- jsPDF + dom-to-image-more
